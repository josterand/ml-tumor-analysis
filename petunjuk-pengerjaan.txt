1. Silahkan Download Dataset
2. Import dataset kedalam program python sesuai dengan nomor kelompok
3. Silakan melakukan Exploratori Data Analysis (EDA) pada datasheet yang sudah ada
4. berikan kesimpulan mengenai dataset yang didapatkan
5. Silahkan lakukan "clustering kepada dataset" + lakukan "Tuning Hyperparameter"
6. Berikan kesimpulan pada hasil clustering
7. Buatlah video presentasi project dan unggahlah ke platform YouTube (silahkan penjelasan dibagi antar anggota kelompok, diusahakan semua orang menjelaskan karena penilaian pribadi diambil dari presentasi)
8. Kumpulkan file project ke dalam 1 folder dengan nama folder Kelas_kelompokN yang berisikan :
--- 1 file PPT mengenai apa yang akan dipresentasikan
--- 1 file Word dengan format (Cover, Pendahuluan [penjelasan mengenai dataset], EDA, Tahap Persiapan, Tahap Preprocessing, Tahap Model Training, Tahap Evaluasi, Kesimpulan dan analisis hasil Machine Learning)
--- 1 file .py / .ipynb
--- 1 file readme.txt berisi link Youtube
9. Kemudian, folder tersebut harus dikompres menjadi file ZIP dengan nama Kelas_kelompokN dan unggahlah ke LMS oleh salah satu anggota kelompok maksimal hari Rabu, 29 April 2026, jam 23.59.
10. Keterlambatan dalam mengumpulkan proyek akan berakibat pada potongan nilai 20%
11. Ketidaksesuaian file dan folder yang dikumpulkan akan berakibat nilai tidak maksimal.

NB :
1) Semakin dalam analisis yang kalian lakukan nilai akan semakin bagus
2) Semakin detail presentasi yang dilakukan nilai akan semakin bagus
3) Hyperparameter yang dilakukan sangat berpengaruh pada nilai
4) Hasil analisis sangat berpengaruh pada nilai
5) Penggunaan AI pada Level 3 hanya digunakan untuk penyelesaian error dan harus dilampirkan sebelum dan sesudah , Jika ketahuan melanggar nilai UTS akan mendapatkan  0
